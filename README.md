# AgriScore Dashboard

AgriScore is a smart farming dashboard providing real-time monitoring, analytics, and AI-assisted guidance for farmers and agricultural organizations.

## Features

- Real-time farm monitoring dashboard
- Crop management and health tracking
- Soil and water analytics using IoT sensor data
- Weather status and forecast integration
- Task management system
- AI-powered chatbot (text and voice) using the Gemini API
- Reports and analytics
- User authentication and profile management

## Technology Stack

- React 19
- TypeScript
- Vite 6
- Tailwind CSS
- Recharts
- Google Gemini API
- React Router
